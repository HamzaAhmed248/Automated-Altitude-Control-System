Automated Altitude Control System
This project is an automated flight stabilizer simulation. Using an HC-SR04 ultrasonic sensor, the system measures ground clearance and uses a Proportional (P) controller to dynamically adjust motor power (PWM).

Features
Ascend & Descend: Automatically adjusts altitude based on a target set via potentiometer (10cm - 30cm).

Smooth Takeoff/Landing: Includes LiftOff() and Land() functions to ramp motor speed safely.

Real-time Feedback: Uses ultrasonic sound waves to calculate distance and correct flight errors instantly.
